# GymSync
Track your powerlifting progress via GymSync! Input your lifts weekly and see the progress as a graph.

File structure:

/GymSync
│
├── app.py
├── templates
│   └── index.html
├── static
│   ├── style.css
│   └── chart.js
└── data.db

While within /GymSync, run '$ python app.py runserver -d' in terminal to launch the application.
